ขับเคลื่อนนวัตกรรม Dyson สู่อนาคต

โซลูชัน High Performance Storage เพื่อรองรับงาน Graphic Design และ R&D ที่เหนือกว่า

ดูข้อเสนอ

สถานการณ์ปัจจุบันและความท้าทาย

ระบบปัจจุบัน (As-Is)

  • ใช้ HPE 3PAR Storage อายุการใช้งาน 7 ปี (Outdated)
  • เพิ่งลงทุนซื้อ Dell GPU Server รุ่นใหม่เมื่อปีที่แล้ว
  • จัดเก็บไฟล์งาน Graphic Design สำหรับแผนก R&D

ความต้องการ (To-Be)

  • ต้องการ High Performance & Low Latency Solution
  • มองหา Data Reduction (เปรียบเทียบกับคำเคลม 5:1 ของ Pure)
  • ต้องการระบบที่เสถียรและคุ้มค่าที่สุด

โซลูชันแนะนำจาก Dell Technologies

เลือกโซลูชันที่ออกแบบมาเพื่อทำงานร่วมกับ Dell GPU Server อย่างสมบูรณ์แบบ

PowerScale F710

สำหรับ Unstructured Data & AI

ที่สุดของ Scale-out File Storage แบบ All-Flash เหมาะสำหรับงาน Graphic, Media และ AI Dataset

Scale-out: ยิ่งเพิ่ม Node ประสิทธิภาพยิ่งสูง
NVMe Performance: รองรับงานไฟล์ขนาดใหญ่
ความจุเริ่มต้น: ~100-300 TB (Cluster 3 Nodes)

PowerStore 5200T

สำหรับ Infra & Scratch Volume

Unified Block & File Storage อัจฉริยะ ที่มีความหน่วงต่ำ (Low Latency) เหมาะกับ VM และ Database

Unified: รองรับทั้ง Block และ File ในตัวเดียว
Data Reduction: การันตี 4:1 (Average) แบบ Always-on
Anytime Upgrade: อัปเกรดคอนโทรลเลอร์ได้โดยไม่หยุดระบบ

ทำไมต้อง Dell Technologies?

ชนะคู่แข่ง (Pure Storage) ด้วย 4 ปัจจัยหลัก

Better Together Ecosystem

Dyson มี Dell GPU Server อยู่แล้ว การเลือก Dell Storage ที่มี Validated Design จะลดความเสี่ยงเรื่อง Compatibility และได้ Performance สูงสุด

Superior TCO & Lifecycle

โปรแกรม Future-Proof และ Lifecycle Extension ช่วยให้การ Refresh ในอนาคตทำได้ง่าย ไม่ต้องทำ Forklift Migration เหมือนคู่แข่ง

End-to-End Support

จุดติดต่อเดียว (Single Point of Contact) ทั้ง Server และ Storage แก้ปัญหาได้เร็วกว่า ไม่มีการโยนความรับผิดชอบระหว่าง Vendor

Flexible Scalability

เริ่มเล็กและขยายได้ตามต้องการ (Start Small, Grow Big) ด้วย PowerScale ที่ยืดหยุ่นกว่า FlashBlade ของคู่แข่งที่เริ่มด้วยขนาดใหญ่และราคาสูง
